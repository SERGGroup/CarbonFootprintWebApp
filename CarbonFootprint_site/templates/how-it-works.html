How it works - brief introduction to CarbonApp

CarbonApp is a calculator that calculates the emissions of Carbon Dioxide equivalent ( because it also evaluates other substances' impact, like methane that contributes to global warming potential) related to the lifestyle of a single person.

In fact it uses data about the lifestyle of the person that uses the calculator; the main fields that we used are: Diet, Transport, Home and Clothes.

All the data that are used for this project are taken by literature and articles, and are all reported into an excel file that you can download at the end of the page.

In each input box you have to insert the related data, if the answer is “0” you can choose to not insert anything.

All the decimal numbers that you insert must use “,” and absolutely not “.” because it will not be evaluated by the program and it will return some errors.

It’s also recommended to insert at least one input for each field, for the correct operation of the site.

For the Diet part you have to insert data about a typical week, and what you eat in average quantities; the most impactful elements are the beef part, so pay more attention to these data in order to get more realistic results.

For the Transport part, you have to similarly insert data about which method to move around in a typical week you use; there is also a part related to airplanes that needs data referred to a typical year, because we supposed that could be easier to insert such data.

Obviously if you use a bicycle or you move by walk you can insert any data in the transport boxes, and you don’t need to insert how much kilometers you run by bicycle/walk because it not produce any emission (probably could be insert the production impact of the bicycle, but this element is negligible with bike sharing). Both walking and bicycling reduce your Footprint and improve your well-being.

For the House part, the first thing to insert is the number of family members, because it allows to distribute the impact of some part, like the impact of house heating or washing machine; the elements that are corrected with the number of family members are indicated into the information box near to the input box.

For the Clothes part, you have to insert the number of clothes that you buy each year; all the data used for this part are affected by lack of knowledge because there is a very poor database about this field, in fact we used data from different countries and this makes the data not regional coerent.

At the end of the input collection you can press the Calculate button in order to see your results.

Different graphs are shown, in particular: there is a comparison of your impact in the four fields that we asked for data, and also a comparison of your global impact with respect to the per capita impact of different countries in the world.

At the end of the results page there is a Download button that allows you to download an excel file that contains all the impacts for each element, and create different and more detailed graphs. Remember to activate the excel file in order to see the results.
